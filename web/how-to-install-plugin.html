How to Install and Use the LegisTracerEU Plugin for Trados Studio 2022 or Newer

1. Download the Plugin

  1. Download the plugin file here: LegisTracerEU.sdlplugin
  2. Close Trados Studio if it is running.
  3. Double-click the downloaded .sdlplugin file OR right-click the file and choose Install (see screenshot below).

2. Manual Installation (if needed)

  1. Make sure hidden files/folders are visible in Windows Explorer (View → Hidden Items).
  2. Copy the downloaded plugin file to:
    C:\Users\USER_NAME\AppData\Roaming\Trados\Trados Studio\17\Plugins\Packages
    (Replace USER_NAME with your Windows account name. "17" is the Studio version and may differ for Studio 2024 or later.)
  3. The plugin only works in Studio 2022 and later versions.

3. Open Trados Studio and Check Plugin Loading

  1. Start Trados Studio.
  2. If you see a message about an unsigned plugin, this should NOT appear (the plugin is signed by RWS). If it does, you are using an outdated version—please download the latest from the link above.

4. Check for LegisTracerEU Button

  1. Open any translation file in Trados Studio.
  2. Go to the Add-ins tab and check if the LegisTracerEU button is visible.

5. If Button Not Visible

  1. Click Plug-in and verify that the LegisTracerEU plugin checkbox is selected.
  2. If not, select it and click OK, then restart Studio.

6. Open the LegisTracerEU Results Panel

  1. Go to View and click LegisTracerEU Results.

7. Confirm the LegisTracerEU Panel is Visible

  1. The panel can be attached to various locations in the Studio UI.
  2. If you see the panel (even if no results are displayed yet), the plugin is loaded correctly.

8. Search in the Panel

  1. In the LegisTracerEU panel, enter a phrase (e.g., resolution tools and powers) in the search field and click Search.
  2. Relevant results should be displayed in the panel.

9. Search from the Editor

  1. In any source segment of the open document, select text and right-click LegisTracerEU Search.
  2. The relevant results will be displayed in the results panel.

10. Set a Keyboard Shortcut (Recommended)

  1. Go to File → Options → Keyboard Shortcuts in Studio.
  2. Locate LegisTracerEU in the list and assign a shortcut (e.g., ALT+Down Arrow).
  3. Now you can select text and use the shortcut to search, instead of using the right-click menu.
Note: The plugin is for Trados Studio 2022 and newer. If you have issues, please check you are using the latest version and follow the manual installation steps if needed. If you need assistance with the installation contact support using the floating Support icon on the bottom right.
